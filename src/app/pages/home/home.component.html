How to take an exam?

  1. Click in 'Exams' tab.
  2. Choose any exam.
  3. After reading the informations, click the button 'Start exam!'.
  4. There are 3 question types:
    • Single choice - only one answer is correct.
    • Multiple choice - at least one answer is correct.
    • Short answer - user enter at least one answer in text input, if there are more than one answer, they have to be separated by comma (,).
  5. Exam ends when time is over, or user finish it by clicking 'Send exam') button.
  6. After finish exam, there will be displayed informations about your score and exam time.

How exam points scoring works?

  1. Single choice question
    • Choosing correct answer: +1 point
    • Choosing incorrect answer: 0 points
  2. Multiple choice question
    • Choosing correct answer: +1 point
    • Choosing incorrect answer: -1 point
    • It is impossible to get less than 0 points for all answers in one question.
  3. Short answer question
    • Answers have to be separated by comma (,) - otherwise, every user answer in an input will be recognized as one answer.
    • Entering correct answer: +1 point
    • Entering incorrect answer: -1 point
    • Entering more answers than there are: -1 point for every useless answer
  4. Keeping answer empty, choosing no one answer will score zero points.

How to add new exam?

  1. Click in 'New Exam' tab.
    1. Fill the Exam name field.
    2. Choose difficulty level from drop down list.
    3. Fill the Exam time field.
      • Entered exam time has to match with HH:MM format (HH - hours, MM - minutes)
      • Exam time cannot be equal to zero!
    4. To add next questions, click in Add question button.
    5. While adding questions and choosing correct answers, questions number and max exam points will be changing.
    6. In depend on question type, there will be added exactly this question type:
      1. Single choice
        • User will get question with answers to choose only one of them.
        • This question type has to contain one, and only one correct answer.
        • This question type has to contain at least two answers.
      2. Multiple choice
        • User will get question with answers, to choose one or more of them.
        • This question type has to contain at least one correct answer.
        • This question type has to contain at least two answers.
      3. Short answer
        • User will get a field, where he can enter his answer.
        • This question type can contain one or more answers.
        • While creating this question type, every answer has to be correct (it will be marked automatically).
    7. To choose which questions are correct, click in square field between answer content and delete button.
    8. To delete question, click in Delete question button, near chosen question.
    9. To delete answer, click in Delete button, near chosen answer.
    10. To save exam in database, after filling fields and adding questions and answers, click in Save exam button. After click, you will get information about the result of saving.
